# Systems Architecture

Real-world agentic system designs built as learning artifacts.

## Case Studies

| System | Domain | Pattern |
|---|---|---|
| [Insurance Claims Processor](insurance-claims-processor/README.md) | InsurTech | Orchestrator + Specialist Sub-agents + HITL |
| [Fashion Email Campaign](fashion-email-campaign/README.md) | RetailTech | Batch Multi-modal Pipeline + RAG + Closed-Loop Feedback |
| [Prior Authorization Workflow](prior-authorization/README.md) | HealthTech | Evidence Aggregation + Policy RAG + Submission Readiness + HITL |

---

Each case study includes:
- Full architecture diagram (ASCII + Mermaid)
- Agent breakdown with tools and responsibilities
- Data flow walkthrough
- Design decision rationale
- Artifact stubs (prompts, tool schemas, state machines) to build from
