# GenAI Learning Curriculum

> Self-paced coverage of the full modern GenAI stack — from foundational language models through fully autonomous agentic systems.

---

## Learning Path

```
01           02            03        04         05          06
LLMs    →  Prompts  →   RAGs   →  MCP    →  Agents  →  Agentic AI

Foundation  Interface  Knowledge  Protocol  Frameworks   Systems
```

- **LLMs** — The engine: a model that predicts the next token
- **Prompts** — The interface: communicating intent to the model
- **RAGs** — The memory: injecting external knowledge at query time
- **MCP** — The protocol: standardizing how tools connect to any LLM
- **Agents** — The actors: LLMs that loop, use tools, take multi-step action
- **Agentic AI** — The systems: multiple agents coordinating autonomously
